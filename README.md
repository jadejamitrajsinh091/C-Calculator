# C-Calculator
# 🧮 C Program – Simple Calculator Using Functions

This project is a simple calculator program written in C language.

It uses:

- Functions
- Switch case
- Loop
- User input

The calculator can perform:

+ Addition  
- Subtraction  
* Multiplication  
/ Division  
% Modulus

## ⚙️ Features

✅ Uses functions for each operation  
✅ Uses switch case  
✅ Uses infinite loop  
✅ Exit option available  
✅ User friendly menu  

---

## ▶️ Menu Example

```
Press 1 for +
Press 2 for -
Press 3 for *
Press 4 for /
Press 5 for %
Press 0 for Exit
```
